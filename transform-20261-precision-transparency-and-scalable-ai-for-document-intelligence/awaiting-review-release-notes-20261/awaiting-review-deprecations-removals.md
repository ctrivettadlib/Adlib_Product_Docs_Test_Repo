# AWAITING REVIEW Deprecations & Removals

This section outlines features and configuration changes that have been removed or are no longer supported in this release.

# Transformation Engine Configuration

Scaler settings have been removed from the Transformation Engine configuration.

## What this means

Scaler settings are no longer managed within the Transformation Engine. If you previously used these settings, you may need to review where scaling is now configured in your environment.

## Background

This change simplifies the Transformation Engine configuration by removing scaler-related settings from this location.
